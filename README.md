# Car Race
A simple car racing game that runs on the browser.

## Build instructions
After cloning the repository, run `yarn` to download the deps, and `yarn start`
to run in your browser.

## How to play
 - A - left lane
 - S - middle lane
 - D - right lane
 - Left arrow - move left
 - Right arrow - move right

## Built with
 - Typescript
 - React
 - Styled Components
